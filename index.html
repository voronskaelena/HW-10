Типы селекторов: по названию тега; по атрибуту; по id; по классу; контекстные селекторы.

Селектор по названию тега: для любого элемента, который можно форматировать. Например, div

По атрибуту: устанавливается стиль элементу, у которого присутствует атрибут. Например, span.

По id: выборка для всех элементов по id.

По классу: применяют если необходимо определить стиль для одного или нескольких элементов. Например, class="box".
